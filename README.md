# CineBook 🎬

**CineBook** is a premium, cinematic cinema booking platform built with a modern MERN stack and a dedicated Java microservice. Designed for a seamless user experience, it blends high-end UI/UX with robust real-time features like live seat locking, immersive 3D theater previews, and professional server-side PDF generation.

---

## ✨ Features

### 🎞️ Immersive Discovery
- **Dynamic Movie Listing**: Fetches the latest movies with TMDB integration.
- **Detailed Movie Profiles**: Insights into ratings, genres, cast, and high-quality posters.
- **Dynamic Showtimes**: Automated schedule generation for multiple theaters.

### 🪑 Advanced Seating Hall
- **Real-Time Seat Locking**: Powered by **Socket.io**; prevents double bookings with a 120-second auto-unlock countdown.
- **3D Hall Preview**: An interactive "Director's View" perspective built with CSS 3D transforms.
- **Neon UI**: Modern glassmorphic theme with neon seat states (Available, Selected, Locked, Premium).

### 🍔 Premium Booking Flow
- **Snack Bar Integration**: Add popcorn, nachos, and beverages to your booking.
- **Wallet & Cashback**: Earn ₹10 cashback per seat; pay using a persistent guest wallet system.
- **Coupon Engine**: Apply codes like `PILU10` or `ENJOY10` for instant discounts.
- **Java PDF Service**: High-fidelity PDF tickets with embedded QR codes generated by a dedicated Spring Boot microservice.

### 📜 User Management
- **Guest Identity**: Persistent guest-user system using `localStorage` for wallet and history tracking.
- **Booking History**: Comprehensive dashboard to view past orders and download tickets at any time.

## 🚀 Getting Started

### Prerequisites
- **Node.js**: v18+
- **Java**: 17+ (for ticket-service)
- **Maven**: (Optional, if not using wrapper)
- **MongoDB**: Local instance or Atlas URI

### 1. Clone the Repository
```bash
git clone https://github.com/bhagyabratagantayat/Cine-Book.git
cd Cine-Book
```

### 2. Setup Backend (Node.js)
```bash
cd backend
npm install
# Create a .env file (see Env Variables section)
npm run dev
```

### 3. Setup Frontend (React)
```bash
cd frontend
npm install
npm run dev
```

### 4. Setup Ticket Service (Java)
```bash
cd ticket-service
# Using Maven Wrapper (recommended)
./mvnw spring-boot:run
```

## 🛣️ API Overview

### Bookings
- `POST /api/bookings` - Create a new booking
- `GET /api/bookings/:id` - Fetch single booking
- `GET /api/bookings/history/:userId` - User history
- `POST /api/bookings/pdf` - Bridge to Java PDF service

### Shows & Movies
- `GET /api/movies` - List active movies
- `GET /api/shows/:id` - Get show layout and theater info

---

## 🗃️ Database Models

- **User**: Identity, persistent wallet balance, and status.
- **Booking**: Central record for movie, theater, seats, QR, and transactions.
- **Show**: Seat layout, locked seats status, and pricing tiers.
- **Food**: Managed snack bar inventory.
- **WalletTransaction**: Debit/Credit history for user cashback.
